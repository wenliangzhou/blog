信息配置

tips

  1. /etc/gitconfig 文件: 包含系统上每一个用户及他们仓库的通用配置。 如果使用带有 —system 选项的
    git config 时，它会从此文件读写配置变量。
  2. ~/.gitconfig 或 ~/.config/git/config 文件：只针对当前用户。 可以传递 —global 选项让 Git
    读写此文件。
  3. 当前使用仓库的 Git 目录中的 config 文件（就是 .git/config）：针对该仓库。
